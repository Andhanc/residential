# Инструкция по запуску проекта Berry Material React Admin Template

## 📋 Требования

Перед запуском убедитесь, что у вас установлены:
- **Node.js** (версия 18 или выше)
- **npm** или **yarn** (менеджер пакетов)

Проверить установку можно командами:
```bash
node --version
npm --version
# или
yarn --version
```

## 🚀 Быстрый старт

### Вариант 1: Полная версия (Full Version)

Полная версия включает все функции, страницы и компоненты - идеально для изучения всех возможностей шаблона.

1. **Перейдите в папку full-version:**
   ```bash
   cd full-version
   ```

2. **Установите зависимости:**
   ```bash
   npm install
   ```
   или если используете yarn:
   ```bash
   yarn install
   ```

3. **Запустите проект в режиме разработки:**
   ```bash
   npm run dev
   ```
   или:
   ```bash
   yarn dev
   ```

4. **Откройте браузер:**
   Проект будет доступен по адресу: **http://localhost:3000**

---

### Вариант 2: Минимальная версия (Seed Version)

Минимальная версия содержит только базовые зависимости и структуру папок - идеально для начала с чистого листа.

1. **Перейдите в папку seed:**

## ⚠️ Возможные проблемы

### Порт 3000 уже занят
Если порт 3000 занят, Next.js автоматически предложит использовать другой порт (например, 3001).

### Ошибки при установке зависимостей
- Убедитесь, что у вас установлена последняя версия Node.js
- Попробуйте удалить папку `node_modules` и файл `package-lock.json` (или `yarn.lock`), затем выполните установку заново:
  ```bash
  rm -rf node_modules package-lock.json
  npm install
  ```

### Проблемы с yarn
Если проект использует yarn, но у вас его нет, установите:
```bash
npm install -g yarn
```
